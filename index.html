L0
무료 · 누구나

입장 — 챗 앱 깔고 로그인

목표: 5분 안에 Claude와 첫 대화.

  • 데스크톱: claude.com/download → macOS(11+)·Windows(10+) 설치 (Linux는 웹 사용)
  • 모바일: App Store(iOS 17+) 또는 Google Play에서 "Claude" 설치
  • 웹: claude.ai 접속 (설치 불필요)
  • Google 또는 이메일로 로그인
첫 질문에 답이 온다
⚠
함정 — 데스크톱 앱은 Linux 미지원. 리눅스라면 웹(claude.ai)으로 시작하세요.
L1
무료

첫 활용 — 능력 켜고 Skill 써보기

목표: 파일을 만들고 스킬로 작업을 자동화.

  • Settings → Capabilities에서 Code execution & file creation 켜기
  • Customize → Skills에서 예시 스킬을 토글
  • (선택) 직접 만든 커스텀 스킬 업로드
스킬로 문서·표를 생성
⚠
함정 — 코드 실행 토글이 꺼져 있으면 커스텀 스킬이 작동하지 않습니다. 커스텀 스킬은 유료(Pro+) 필요.
L2
무료~

연결 — Connector(MCP) 붙이기

목표: 캘린더·노션 같은 내 도구를 Claude에 연결.

  • claude.ai/directory에서 검수된 커넥터 둘러보기
  • Settings → Connectors → “+” Add custom connector → 원격 MCP 서버 URL 입력
  • 채팅 입력창 “+”에서 커넥터를 켜고 사용
채팅이 외부 도구를 호출
⚠
함정 — 무료는 커스텀 커넥터 1개까지. 모바일은 직접 추가 불가(웹/데스크톱에서 추가 → 동기화, beta). 신뢰할 수 없는 서버는 연결 금지(prompt injection 위험).
L3
유료 · Pro+

개발자 진입 — Claude Code 설치

목표: 터미널·IDE에서 코딩 에이전트 사용.

$curl -fsSL https://claude.ai/install.sh | bash
>irm https://claude.ai/install.ps1 | iex
  • 설치 후 첫 실행 claude → 브라우저 로그인
  • IDE: VS Code(1.98+) “Claude Code” 확장 / JetBrains 플러그인(ID 27310, Beta)
claude 세션이 시작된다
⚠
함정 — 무료 플랜 불가(Pro $17/mo~). IDE 확장만으론 터미널 claude가 없습니다 → CLI를 별도 설치. Windows 데스크톱 Code 탭은 Git for Windows 필요. ANTHROPIC_API_KEY가 있으면 구독 대신 토큰 과금됨.
L4
유료 · 파워

파워업 — MCP · Skills · Plugins 무장

목표: 나만의 도구로 Claude Code를 확장.

MCP 서버 추가

$claude mcp add --transport http <name> <url>

공식 Skill 마켓 추가 후 설치

$/plugin marketplace add anthropics/skills

플러그인 설치 (Claude Code 전용)

$/plugin install <name>@<marketplace>
플러그인·스킬·MCP가 세션에서 작동
⚠
함정 — 확장은 당신 권한으로 코드를 실행합니다. 신뢰하는 출처만. Plugins는 Claude Code 전용(앱·웹 기능 아님). 더 깊게: 확장 추천 →

설치 전 1분

확장 = 코드 실행. 신뢰부터.

Skill · MCP 서버 · Plugin은 당신의 권한으로 코드를 실행하고 파일·도구에 접근할 수 있습니다. 편리한 만큼, 아무거나 설치하면 위험합니다. 설치 전 3가지만 확인하세요.

FAQ

꼭 돈을 내야 하나요?
챗 앱(웹·데스크톱·모바일)은 무료입니다. 코딩 에이전트인 Claude Code만 유료(Pro 연납 월 $17~). Skill·커넥터는 무료에서도 일부 됩니다.
Linux도 되나요?
데스크톱 앱은 Linux를 지원하지 않습니다. 웹(claude.ai)을 쓰거나, 개발자라면 Claude Code CLI(네이티브 설치)를 사용하세요.
모바일에서 다 되나요?
채팅과 커넥터 사용은 됩니다. 단 커스텀 커넥터 추가는 웹/데스크톱에서 하고 모바일로 동기화됩니다(모바일 추가는 beta).
VS Code/JetBrains 확장만 깔면 되나요?
아니요. IDE 확장은 채팅 패널용 CLI만 내장하고 터미널의 claude를 깔아주지 않습니다. claude mcp add 같은 명령을 쓰려면 standalone CLI를 별도 설치하세요.
Skill · MCP · Plugin, 뭐가 다른가요?
Skill = 능력 폴더(모든 표면에서 사용) · MCP/Connector = 외부 도구 연결(표면마다 추가법 다름) · Plugin = 여러 개를 묶은 패키지(Claude Code 전용). 무엇부터 깔지는 확장 추천에서.
